# Story 002: Avatar Chat UI

## User Story
As the user, I want Donna represented by an avatar beside a chat window, so that the assistant feels present and easy to interact with.

## Acceptance Criteria
- The avatar is shown on the left.
- The chat and information area is shown on the right.
- The chat area height matches the avatar height.
- The chat area width is 80% of the avatar height by default.
- Donna messages align left.
- User messages align right.
- The bottom area contains the chat input.
- The top-left of the input area shows Donna's current state.
- The top-right of the input area shows the selected chat model.
- Message colors for Donna and the user are configurable.

## Notes
- The UI must be styled carefully with good spacing, readable font sizes, and aligned elements.
