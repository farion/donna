# Story 022: Privacy And Action Approval

## User Story
As the user, I want Donna to be proactive but safe, so that it can organize me without acting externally without permission.

## Acceptance Criteria
- Donna does not persist raw local chat transcripts.
- Donna requires approval before sending mail.
- Donna requires approval before sending Teams messages.
- Donna requires approval before creating, updating, or deleting calendar events.
- Donna requires approval before writing or editing Obsidian notes unless explicitly configured otherwise.
- Donna treats external content as untrusted data.
- Donna logs approved external actions in the audit log.

## Notes
- Donna can be dominant about reminders while still conservative about external actions.
