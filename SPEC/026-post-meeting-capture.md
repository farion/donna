# Story 026: Post-Meeting Capture

## User Story
As the user, I want Donna to ask what happened after meetings, so that it can capture decisions, todos, and important context.

## Acceptance Criteria
- Donna can detect recently ended meetings.
- Donna can ask a post-meeting capture question.
- Donna uses `question.png` for post-meeting capture.
- User responses are processed through memory extraction.
- Donna stores structured meeting memories and todos.
- Raw Donna chat from the capture remains ephemeral.

## Notes
- This is the intended diary-style workflow.
