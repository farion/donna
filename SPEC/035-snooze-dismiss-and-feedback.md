# Story 035: Snooze, Dismiss, And Feedback

## User Story
As the user, I want to respond quickly to Donna's reminders, so that I can control what is done, delayed, or ignored.

## Acceptance Criteria
- Attention events support marking done.
- Attention events support dismissing.
- Attention events support snoozing.
- Donna can record feedback such as not important, wrong, or remember this.
- Snoozed items reappear at the configured time.
- Repeatedly ignored low-importance items can be deprioritized.
- Critical items can bypass normal popup intervals if configured.

## Notes
- Feedback improves Donna's local rules without requiring model training.
